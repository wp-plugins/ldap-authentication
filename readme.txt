Authenticate against an LDAP server

== Description ==

Once installed, this plugin will authenticate against an LDAP server (Tested on Windows Server 2003).


== Usage ==

Fill in the details found under the admin menu tabs and login.
An OU can be band like a Student OU.
If activated, plugin will search LDAP and if the user is not found then it will search your wordpress database.


== Installation ==

- Upload `plugin-name.php` to the /wp-content/plugins/ directory
- Activate the plugin through the 'Plugins' menu in WordPress
- Fill in the required details on the 'Admin Menu'
